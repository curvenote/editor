prosemirror-codemark

prosemirror-codemark is a plugin for ProseMirror that handles manipulating inline code marks. The plugin creates a fake cursor as necessary to show you if the next character you type will or will not be marked. The plugin is created and used by Curvenote.

Source code and documentation are on GitHub.
